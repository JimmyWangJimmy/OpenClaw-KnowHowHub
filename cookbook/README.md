# OpenClaw Cookbook

Status: Verified
Last Updated: 2026-03-21
Source: repository curation

OpenClaw Cookbook 面向“我现在就要做一件事”的读者。

它不讲抽象原则，而是直接给步骤、配方、默认顺序和可复用模板。

## 这本手册帮你做什么（This Book Helps You）

| 你想做... | 打开这页 |
| --- | --- |
| 尽快跑起来 | [Quick Start Recipe](quick-start.md) |
| 先把安全基线打好 | [Safe Setup Recipe](safe-setup.md) |
| 把渠道接起来 | [Channel Setup Recipe](channel-setup.md) |
| 安装和试用 skills | [Skill Installation Recipe](skill-installation.md) |
| 写第一套 playbook | [Build Your First Playbook](first-playbook.md) |

## 覆盖范围（What This Book Covers）

- 最短上手路径
- 默认安全顺序
- 渠道与技能的常见起步动作
- 第一套可复用 playbook 的搭建方式
