Status: Verified

## Purpose

本页是 OpenClaw 生态入口，优先收录高质量、仍可访问、对实际使用有帮助的来源。

## Selection Rules

- 优先一手来源、长期维护来源、核心社区来源
- 优先仍在 2025-2026 年有更新痕迹的项目
- 不收明显蹭关键词的无关 “OpenClaw AI” 页面
- 不在本仓库镜像受版权限制的游戏资源，只保留说明和入口

## Core Community

### The Claw Recluse

- Link: <https://captainclaw.net/en/>
- Type: 社区门户 / 下载 / 新闻 / 历史归档
- Why it matters: 当前最核心的 Captain Claw 社区入口之一。站点介绍写明它始于 2004 年，现已覆盖二十多年社区积累，包括自定义关卡、地图编辑器、同人作品和速通记录。
- Best for: 新用户、回坑玩家、找下载包和社区动态的人
- Verified on: 2026-03-01

### The Claw Recluse Downloads

- Link: <https://captainclaw.net/en/downloads.html>

- Best for: 想快速开始玩、找现代系统兼容包、找关卡编辑工具的人
- Verified on: 2026-03-01

## Code And Tools

### pjasicek/OpenClaw

- Link: <https://github.com/pjasicek/OpenClaw>
- Type: 开源代码仓库
- Why it matters: 目前最核心的 Captain Claw 开源重实现之一。仓库明确说明这是一个跨平台 C++ 重实现，代码从零编写，并且支持 Windows、Linux、Android 和 Emscripten/WebAssembly 构建路径。
- Best for: 想研究引擎实现、移植、构建流程和资源结构的人
- Verified on: 2026-03-01

### Zax37/WapMap

- Link: <https://github.com/Zax37/WapMap>
- Type: 地图编辑器
- Why it matters: 这是当前最值得优先关注的关卡编辑器仓库。README 明确说明它是 fanmade WAP32 编辑器，最常用于 Claw 地图编辑；仓库显示最新版本 `v0.5.0 (b39)` 发布于 2026-01-18。
- Best for: 自定义关卡设计、地图研究、编辑器二次开发
- Verified on: 2026-03-01

### WapMap Releases

- Link: <https://github.com/Zax37/WapMap/releases>
- Type: 版本发布页
- Why it matters: 这里可以直接看见编辑器的更新速度和近期改动方向。2026 年初的版本迭代仍然频繁，说明项目仍处于活跃维护状态。
- Best for: 跟踪版本、下载最新编辑器、判断项目活跃度
- Verified on: 2026-03-01
