# Tool List

Status: Draft
Last Updated: 2026-03-01
Source: curated web research

## Goal

本页集中整理当前仓库里反复出现的核心工具和项目，帮助读者快速判断“我需要哪一个”，而不是在不同文档之间来回找同一个名字。

## Core Runtime And Community Tools

### CrazyHook

- Primary source: <https://captainclaw.net/en/downloads.html>
- Role: 现代社区默认游玩与增强基线
- Best for: 在现代系统上游玩、使用社区默认增强功能、运行许多依赖当前生态的内容

## Core Editing Tools

### WapMap

- Project: <https://github.com/Zax37/WapMap>
- Releases: <https://github.com/Zax37/WapMap/releases>
- Role: 当前主线关卡编辑器
- Best for: 自定义关卡制作、现代地图编辑工作流、关卡研究

### WapWorld

- Legacy docs: <https://captainclaw.net/es/edytor.html>
- Role: 历史编辑器与旧工作流背景
- Best for: 阅读旧教程、理解旧语境、研究 `.WWD` 相关历史资料

## Core Code And Research Projects

### OpenClaw

- Project: <https://github.com/pjasicek/OpenClaw>
- Role: 开源跨平台重实现
- Best for: 研究实现、构建流程、跨平台路径、源码层分析

## How To Choose Quickly

| If you want to... | Start with... |
| --- | --- |
| 现代系统稳定游玩 | CrazyHook |
| 制作或修改关卡 | WapMap |
| 阅读旧教程和旧编辑语境 | WapWorld |
| 研究实现和源码 | OpenClaw |
