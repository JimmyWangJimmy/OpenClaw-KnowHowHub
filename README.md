# OpenClaw KnowHowHub

OpenClaw / Captain Claw 的公共知识入口。这个仓库的目标不是简单堆资料，而是把分散的教程、参考资料、研究记录和社区资源整理成一个可查找、可复用、可协作的知识库。

如果你希望这个生态更容易被发现、被理解、被延续，这个仓库就是为此而建。

## What You Can Find Here

- 新手入门说明：从哪里开始了解 OpenClaw
- 操作指南：安装、资源提取、基础修改、常见问题排查
- 参考资料：文件格式、工具说明、行为记录
- 研究记录：实验、逆向、待验证发现
- 社区资源：相关项目、链接、协作方向

## Why This Repository Exists

- OpenClaw / Captain Claw 的高质量资料长期分散在旧站点、下载页、工具仓库和社区角落里
- 新人很难快速分清哪些内容仍然可用，哪些只是历史痕迹
- 这个仓库的职责是把高价值入口整理出来，让更多人更快进入生态

## Why Star This Repo

- 你会得到一个持续整理中的高质量入口，而不是零散书签
- 你可以用它快速找到安装、关卡制作、技术研究和社区入口
- Star 也是在给这个生态一个公开信号：这些资料值得继续整理和维护

## Repository Structure

```text
docs/        核心入口文档，适合第一次访问仓库的人
guides/      可操作教程，强调步骤和结果
reference/   查阅型资料，例如格式、配置、行为说明
research/    研究过程、实验记录、逆向分析和待验证结论
community/   相关项目、外部链接、社区协作信息
assets/      截图、示意图、示例资源
tools/       脚本、转换工具、辅助程序
```

## Content Status

为了让内容可信、可维护，建议每篇文档在开头标明状态：

- `Draft`: 初稿，可能不完整
- `Verified`: 已验证，可作为稳定参考
- `In Progress`: 正在整理或研究中
- `Deprecated`: 过时内容，保留仅供历史参考
