## What This Repo Is

这不是普通资料库，也不是零散链接收藏夹。

这是一个同时服务两类读者的 OpenClaw 必读入口：

- `Red Book`: 讲原则、方法论、边界和判断标准
- `Cookbook`: 讲步骤、配方、模板和实战入口
