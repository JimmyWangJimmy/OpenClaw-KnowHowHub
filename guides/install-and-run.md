## Goal

如果你的目标是尽快在现代系统上运行 Captain Claw / OpenClaw 相关内容，这一页给出最短路径。

## Recommended Path

### 1. Start With The Community Downloads Page

- Link: <https://captainclaw.net/en/downloads.html>
- Why first: 这是当前最集中的可用下载入口，包含现代兼容版本、完整关卡包和关键工具。

### 2. Read Technical Support Before Troubleshooting Blindly

- Link: <https://captainclaw.net/en/support.html>
- Why second: 现代系统上最常见的问题并不是“游戏坏了”，而是兼容层、DirectPlay、帧率、权限和配置问题。这个页面已经把高频问题整理好了。

## Common Scenarios

### I just want to play the game

1. Open the downloads page
2. Prefer the current CrazyHook-based package when appropriate
3. If something fails, check technical support before trying random fixes

### The game does not launch correctly on modern Windows

1. Open technical support
2. Check DirectPlay, `cnc-ddraw`, save permissions and frame-rate related notes
3. Only then try compatibility settings or manual patching

### I want custom levels too

1. Start at the downloads page
2. Look for the complete level pack and tool references
3. Use the custom levels directory after the base game is stable
