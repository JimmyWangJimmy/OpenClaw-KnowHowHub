# Level Editing

Status: Draft
Last Updated: 2026-03-01
Source: curated web research

## Goal

如果你的目标是开始制作或研究自定义关卡，这一页给出当前最合理的入门顺序。

## Recommended Path

### 1. Understand The Current Tool Baseline

- WapMap project: <https://github.com/Zax37/WapMap>
- WapMap releases: <https://github.com/Zax37/WapMap/releases>
- Why first: 这是当前主线编辑器。与其从过时工具开始，不如先理解今天社区实际在用什么。

### 2. Read The Level Designer's Guide

- Primary source: <https://captainclaw.net/en/downloads.html>
- Why second: 下载页明确说明有完整的八部分设计指南，这比零散帖子更适合作为系统入门。

### 3. Study Existing Custom Levels

- Link: <https://captainclaw.net/en/clevels.html>
- Why third: 最快的学习方式之一不是“空白开做”，而是研究已有作品，特别是被推荐的关卡、需要 CrazyHook 的关卡和带视频说明的项目。

## Tooling Notes

### WapMap vs. WapWorld

- If you want the current workflow: start with WapMap
- If you are reading old tutorials or studying historical content: use the legacy WapWorld documentation as background only
- Legacy documentation: <https://captainclaw.net/es/edytor.html>

## Suggested Learning Sequence

1. Install and open WapMap
2. Read the Level Designer's Guide
3. Download and inspect several custom levels
4. Recreate a small section before attempting a full original map
