# 配方手册（Cookbook）

配方手册（Cookbook）聚焦执行落地：

- 最短上手路径
- 安全默认顺序
- 渠道与技能接入
- 把重复任务沉淀成流程（playbooks）

3. [Channel Setup Recipe](https://github.com/JimmyWangJimmy/OpenClaw-KnowHowHub/blob/main/cookbook/channel-setup.md)
4. [Skill Installation Recipe](https://github.com/JimmyWangJimmy/OpenClaw-KnowHowHub/blob/main/cookbook/skill-installation.md)
5. [Build Your First Playbook](https://github.com/JimmyWangJimmy/OpenClaw-KnowHowHub/blob/main/cookbook/first-playbook.md)

## 一句话原则（One-Line Rule）

先把一条稳定路径跑通，再渐进扩展渠道、技能和自动化。
