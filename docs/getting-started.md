## Purpose

这个仓库用于整理 OpenClaw 的可复用知识，帮助不同读者在最短路径内找到可执行内容。

## Who This Repository Is For

- 第一次接触 OpenClaw 的新用户
- 已在使用 OpenClaw，想提升稳定性与安全性的操作者
- 想写 skills、playbooks、规则和长期工作流的高级用户
- 需要做来源追踪、版本信号分析和对照研究的研究者
