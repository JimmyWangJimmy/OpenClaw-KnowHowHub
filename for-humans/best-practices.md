# Best Practices

Status: Draft
Last Updated: 2026-03-01
Source: repository design

## Goal

面向人类总结使用 OpenClaw 时最值得优先建立的习惯。

## Core Practices

- 先定义任务目标，再让 agent 行动
- 把模糊请求转成可执行结果
- 给出足够上下文，但不要灌太多无关资料
- 用结构化目录和 playbook 降低重复劳动
- 让 agent 处理重复流程，人类保留方向判断

## Common Failures

- 把 OpenClaw 当成万能聊天机器人
- 没有输入边界，却期待稳定输出
- 让它同时追多个目标，结果信息结构崩掉
