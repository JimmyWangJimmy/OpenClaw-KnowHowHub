# Playbooks

Status: Verified
Last Updated: 2026-03-01

这一层收录面向具体任务的执行手册。

## Typical Playbooks

- 代码审查
- 文档维护
- 来源搜集
- Issue triage
- PR 修复
- 技能创建与维护

## Why Playbooks Matter

- Official source: <https://docs.openclaw.ai/prose>
- Why it matters: OpenProse 的官方定位本身就是 repeatable workflows 和 reusable programs。这说明复杂任务最适合被沉淀成 playbook，而不是一次次重新 prompt。

## What A Good Playbook Should Contain

- 任务前提
- 权限和风险边界
- 执行步骤
- 检查点
- 预期交付物
- 失败恢复路径

## Best First Playbooks To Build

- 来源搜集与筛选
- 文档更新
- 安全审计与风险复查
- Issue triage
- 代码改动前的上下文扫描
